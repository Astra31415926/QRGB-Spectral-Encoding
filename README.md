# QRGB-Spectral-Encoding (v4.5)

**A browser-based tool for generation and spectral analysis of high-capacity 2D color codes.**

---

### 🌐 [OPEN LIVE DEMO](https://astra31415926.github.io/QRGB-Spectral-Encoding/)

---

### 🔬 Project Vision & RSS Method
This website is designed as an **interactive experimental sandbox** for testing and developing the **RSS (Reflectance Synchronized System)** method. 

**Important Disclaimer:** Currently, this tool is intended for **digital testing** of RGB-channel data integrity. It does **not** automatically prepare codes for stable physical printing yet. The next stage of development focuses on implementing RSS mathematical algorithms to compensate for spectral **cross-talk** (color bleeding) in physical media.

### 🚀 Key Features
- **RGB Multiplexing:** Encode independent text data into 3 color channels (Red, Green, Blue) simultaneously, tripling data density.
- **Spectral Scanner:** Real-time decomposition of uploaded images/camera feeds into separate channels for data extraction.
- **Web Audio Feedback:** Generative sound response upon detecting valid data within specific spectral layers.
- **RSS Logic Foundation:** Brightness synchronization for improved channel separation and decoding stability.
